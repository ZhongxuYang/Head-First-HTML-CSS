5 增加图像

 一、<img src="../images/green.jpg">

 二、使用alt属性为图像添加提示（方便视力障碍的人使用）：

  <img src="../images/green.jpg" alt="这是一张绿色的图片">

 三、可以使用width和height属性调整图像的大小：

  1、当然最好不要这么做，调整浏览器中的图像大小已经属于表现的范畴，所以最好是使用CSS来调整。

  2、而且，当使用width和height调整图像大小时，浏览器下载的仍然是大图像，浪费资源。

  3、width和height属性实际上是帮助浏览器确定要为这个图像预留多大的空间的作用。蔬、如果要使用这两个属性，则他们应该与图像的实际宽度和高度一致。

 四、JPEG、PNG和GIF的区别：

  1、JPEG：可表示1600万种色彩。有损、不支持透明度、不支持动画、文件小。

  2、PNG：可表示上百万种色彩。无损、支持透明、适合单色图像和线条图像LOGO/剪贴画等、文件大一些。

  3、GIF：可表示256种色彩。无损、支持透明（只允许一种颜色透明）、支持动画、适合单色图像和线条图像LOGO/剪贴画等、文件大一些。

 五、善于在主页面用缩略图，可以点击缩略图之后链接一个大图像。因为图像太大，主页打开得会很慢。

 六、把图像变成链接：

  <a href="html/seattle_downtown.html"> <img src="../images/green.jpg" alt="这是一个绿色的图像"> </a>

 七、把一个透明图像放在WEB页面中，要确保图像的蒙版颜色与WEB页面的背景色一致：

  设置方法：在Photoshop中，使用“保存为web”对话框中的Matte（蒙版）菜单来选择适合的颜色，并且用反锯齿技术来柔化图像边缘。

 十、使用多个样式表：

  2、内边距、边框、外边距都是可选的。

  5、边框宽度也可以用像素设置，也可以使用关键词thin/medium/thick来指定。

  6、如果两个样式表包含冲突的属性定义，HTML文件中最后链接的样式表最为优先。

  7、可以在<link>元素中使用媒体查询或者使用CSS中的@media规则来指定设备。

11 布局与定位

 一、浮动布局：float:right;

  1、关于float：float属性首先尽可能远的向左或者向右浮动一个元素。然后它下面的所有内容会绕流这个元素。

  2、宽度：width:200px;

  3、“流”的定义。

  4、当浏览器并排放至两个元素时：如果左边的元素外边距为10px，右边的元素外边距为20px，那么这两个元素之间就会有30px的空间。

  5、当浏览器上下放置两个元素时：会折叠他们的外边距。（他们共同的外边距为两个外边距中较大的那么外边距的大小）

  6、内联元素也可以有外边距。不过通常不会设置内联元素的外边距，只有一个例外，就是图像。对于图像，通常不仅会设置外边距，还会设置内边距和边框，这是很常见的。

  *如何设置一个浮动元素：

   1、首先指定一个标识（id）。

   2、然后指定一个宽度（width）。

   3、最后设置浮动。（float：left或right；）

   *浮动一个元素时，如果希望他在摸个元素的后面，就要移动浮动元素的html，让他紧挨着要放得元素的后面。

  7、通过设置外边距（margin：）来解决浮动元素与流的重合问题。

  8、解决浮动元素与页脚的重叠问题用“clear:right或left;”

  9、运用伪类：为链接的不同状态指定样式，如：

   a:link { color:#b76666; text-decoration:none; border-bottom:thin dotted #b76666; }

   a:visited { color:#675c47; text-decoratin:none; border-bottom:thin dotted #675c47; }

 --------------------------------------------------------------------------------------------------------------------*在此之前，我们所采用的所有设计都称为“流体布局”。因为不论我们将浏览器调整到多大的宽度，布局都会拓展，填满整个浏览器。*----------------------------------------------------------------------------------------------------------------

 二、流体与冻结设计：

  1、HTML修改：在html中要增加一个新的<div>元素，并指明id。顾名思义，这个元素要包含页面中的所有内容。

  2、CSS修改：我们要限制这个id的<div>中所有的元素和内容大小，使他们有一个固定的宽度。（也可设置内边距/背景色等）

  *经过以上的设计，就叫“冻结布局“，不论浏览器怎么调整大小，他都不会再移动。

 三、凝胶布局（是流体和冻结之间的状态）：

  *凝胶布局会锁定页面中内容区的宽度，不过会将它在浏览器中居中*

  设置方法首先有一个固定大小的<div>包围页面中的所有内容，然后只需在CSS中的id{ }中加入：margin-left:auto;和margin-right:auto;即可

 四、绝对布局：（一个元素绝对定位时，浏览器首先要做的是将它从流中完全删除，然后浏览器将这个元素放置在top/bottom和right/left属性指定的位置上）

  1、首先我们使用position:absolute;属性指定这个元素要绝对定位。

  2、接下来设置top/bottom和right/left属性。

  3、最后为<div>设置一个宽度。

  *流中的元素不会将其内联元素内容围绕在一个绝对定位元素周围。他们完全不知道页面上有这个绝对定位的元素。

  *谁在上面？

   每个定位元素都有一个名为z-index的属性，这会指定它在一个虚拟z轴上的位置（上面的元素“更靠近”你，z-index更大）。

 五、CSS表格显示：

  1、注意：由于我们在表格中只能放置块元素，所以要把内容包围在<div>中。

  2、表格会自动扩展来适应单元格宽度和高度。

  3、如何创建CSS和HTML实现表格显示：

   不难想到，我们需要增加CSS，告诉浏览器要把我们的栏目显示的像一个表格，不过还需要增加一些HTML。为什么？我们需要增加一些结构来表示行和列，还要增加这个表格本身的结构。

   *做法很简单，只需要为整个表格创建一个<div>，然后为每一行分别创建一个<div>。对于每一列，只需要在行<div>中放置一个块级元素*

   （1）、为表格显示增加HTML结构：

    ①、首先创建一个<div>表示整个表格，行和列要嵌套在这个<div>中。

    ②、接下来，对于表格中的每一行，要创建一个<div>，其中包含行内容。

    ③、然后，对于每一列，只需要一个块元素作为该列内容。

   （2）、为表格创建CSS样式：

    ①、首先，为整个表格<div>增加样式：

     display: table;

    ②、接下来为每一行<div>增加样式（如果有多行，则需要多个行<div>）：display: table-row;

    ③、最后，为每一列<div>指定样式：display: table-cell;

     *在列的CSS中，可以加入vertical-align:top;/middle;/bottom;来设置与单元格上边的对齐方式。如果不设置，默认为中间对齐。

     *可以使用width属性来控制列的宽度。（最好使用百分数的方式，因为使用百分数表格可以根据浏览器的大小自动调整扩展和收缩。且最好确保所有列宽度加在一起为100%）

 六、关于布局的总结

  1、浮动布局：

   ①、关键词：float、clear

   ②、缺点：这种方法有一个很大的缺点，我们必须把需要浮动的整个<div>移动到主页面之上，如果这种顺序并不反映页面内容的相对重要性，这种方法往往是最不优的。另外还有一个缺点，就是使用float无法创建两个高度相同的列。

  2、凝胶布局：

   ①、关键词：auto

   ②、缺点：在于内容不会扩展来填满整个浏览器窗口。（不过很多人认为这不是一个缺点，你会看到有很多博客和Web页面都会采用这种凝胶设计）

  3、绝对布局：

   ①、原理：通过将边栏设置为一个特定的宽度，并将它固定在主内容右边，就有了一个可以岁页面大小扩展的收缩的主内容区，而边栏大小可以一直保持固定。

   ②、缺点：浏览器变宽时，边栏会再次覆盖页脚。

  4、表格显示布局：

   ①、不是最佳选择，但却表现完美。

 七、固定定位：

  1、方法：position:fixed; top/bottom:**px; right/left:**px;

  2、如何工作：一旦采用固定定位方式放置内容，他就会一直留在你原先指定的位置，不再移动，即使你滚动页面他也原地不动。

12 HTML5标记·现代HTML

 一、新元素：

  *把一些没有意义的<div id="**">换为更加有结构意义的元素

  1、<header>：页眉

  2、<footer>：页脚

  3、<nav>：导航

  4、<section>：区段

  5、<article>：定义文章 / 外部内容

  6、<aside>：边栏

  7、<time>：定义时间和日期

   ①、如果元素内容没有采用官方Internet日期/时间格式来写，就必须有datetime属性。

   ②、datetime属性：<time datetime="2012-02-18">2/18/2012</time>

  *本章用到的HTML5新元素，在一些浏览器中可能并不支持。智能手机上会全部支持这些新元素。
